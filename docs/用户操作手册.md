# 生产任务排程管理系统 - 用户操作手册

## 1. 系统简介

生产任务排程管理系统是专为制造工厂设计的智能排程平台，通过甘特图可视化方式展示各生产订单在不同机器设备上的时间安排，支持多用户协同排程，并具备实时的时间重叠与资源冲突检测机制。

### 1.1 核心功能
- 📋 **订单管理**：生产订单的创建、编辑、查询和删除
- 📊 **甘特图排程**：可视化拖拽排程，直观展示机器负载
- ⚠️ **冲突检测**：实时检测时间重叠，杜绝超排现象
- 👥 **多用户协同**：支持10+计划员同时在线操作
- 📈 **数据统计**：产能分析、负载监控、订单状态追踪

### 1.2 用户角色
| 角色 | 权限说明 |
|------|---------|
| 系统管理员 | 完整权限，用户管理、系统配置、数据备份 |
| 计划员 | 订单管理、甘特图排程、数据查看 |
| 查看者 | 订单查询、甘特图查看、报表导出 |

### 1.3 系统要求
- 浏览器：Chrome 90+ / Firefox 88+ / Edge 90+ / Safari 14+
- 屏幕分辨率：推荐 1920×1080 及以上
- 网络：稳定的网络连接（响应时间 < 300ms）

## 3. 订单管理模块

### 3.1 查看订单列表

1. 点击左侧导航栏「📋 订单」
2. **筛选功能**：
   - 搜索框：输入订单号或产品名称搜索
   - 状态筛选：全部/待排程/已排程/生产中/已完成/已取消
   - 优先级筛选：全部/低/中/高/紧急
3. **排序**：点击表头可按该列排序
4. **分页**：底部可切换页码和每页显示数量

### 3.2 新增订单

1. 点击「+ 新增订单」按钮
2. 填写订单信息：
   | 字段 | 必填 | 说明 |
   |------|------|------|
   | 订单号 | ✅ | 如：ORD-2024-0001，系统可自动生成 |
   | 产品 | ✅ | 从下拉列表选择产品 |
   | 生产数量 | ✅ | 正整数 |
   | 交付日期 | ✅ | 选择生产日期 |
   | 优先级 | ✅ | 低/中/高/紧急 |
   | 备注 | ❌ | 订单相关说明 |
3. **自动计算**：系统自动计算预估工时 = 数量 × 产品工时
4. 点击「保存」完成创建

### 3.3 编辑订单

1. 在订单列表中找到要编辑的订单
2. 点击操作列「✏️ 编辑」按钮
3. 修改需要更新的字段
4. 点击「保存」

> ⚠️ 注意：已排程的订单修改后需要重新排程。

### 3.4 删除订单

1. 点击操作列「🗑️ 删除」按钮
2. 确认删除提示
3. 删除成功

> ⚠️ 注意：已排程的订单删除后，对应的排程也会被自动删除。

### 3.5 快捷排程

1. 在订单列表中找到待排程订单
2. 点击操作列「📅 去排程」按钮
3. 系统自动跳转到甘特图页面，并选中该订单

## 4. 甘特图排程模块

### 4.1 甘特图界面介绍

```
┌─────────────────────────────────────────────────────────────────────────┐

└──────────────┴──────────────────────────────────────────────────────────┘
```

### 4.2 视图操作

1. **切换视图**：
   - 日视图：显示每天 24 小时时间轴
   - 周视图：显示每周 7 天时间轴

2. **缩放控制**：
   - 点击「-」缩小（最小 50%）
   - 点击「+」放大（最大 200%）
   - 拖动滑块精细调整

3. **时间范围**：
   - 点击日期范围选择器
   - 选择开始日期和结束日期
   - 点击「确定」刷新甘特图

4. **滚动操作**：
   - 横向滚动：查看不同时间段
   - 纵向滚动：查看更多机器

### 4.3 创建排程（拖拽方式）

1. **选择订单**：在左下角「待排程订单」面板中找到要排程的订单
2. **开始拖拽**：按住鼠标左键拖动订单
3. **选择机器**：拖动到目标机器行
4. **选择时间**：移动到目标开始时间位置
5. **释放鼠标**：完成排程创建
6. **自动调整**：系统自动根据预估工时设置结束时间

> 💡 技巧：拖拽时按住 Shift 键可以精确对齐到小时刻度。

### 4.4 调整排程时间

**移动整个排程：**
1. 将鼠标移到任务块中间
2. 按住鼠标左键拖动
3. 上下拖动可更换机器
4. 左右拖动可调整开始时间
5. 释放鼠标完成移动

**调整开始时间：**
1. 将鼠标移到任务块左边缘（光标变为 ↔）
2. 按住鼠标左键拖动
3. 向左拖动提前开始时间
4. 向右拖动延后开始时间

**调整结束时间：**
1. 将鼠标移到任务块右边缘（光标变为 ↔）
2. 按住鼠标左键拖动
3. 向左拖动缩短生产时间
4. 向右拖动延长生产时间

### 4.5 冲突检测机制

**实时检测：**
- 拖拽过程中系统实时检测时间冲突
- 无冲突时：任务块显示正常颜色（根据优先级）
- 有冲突时：任务块边框变红并闪烁

**冲突提示：**
- 鼠标悬停在冲突任务块上显示详情
- 右侧属性面板显示冲突订单列表
- 包含订单号、产品名称、冲突时间段、重叠分钟数

**处理方式：**
- 冲突的排程无法保存
- 需要调整时间避开冲突
- 或与其他计划员协调调整现有排程

### 4.6 任务详情

1. 点击任意任务块
2. 右侧弹出详情面板
3. 查看信息：
   - 订单信息：订单号、产品、数量、交付日期
   - 排程信息：机器、开始时间、结束时间、实际工时
   - 优先级标签
4. 可操作：
   - 手动调整时间（精确到分钟）
   - 删除排程
   - 标记开始生产
   - 标记生产完成

### 4.7 颜色说明

| 颜色 | 优先级 | 说明 |
|------|--------|------|
| 🔴 红色 | 紧急 | 必须优先处理，尽快排程 |
| 🟠 橙色 | 高 | 重要订单，优先安排 |
| 🔵 蓝色 | 中 | 普通订单，正常排程 |
| 🟢 绿色 | 低 | 备货订单，灵活安排 |

**状态颜色：**
- 绿色边框：已排程
- 蓝色边框：生产中
- 灰色边框：已完成
- 红色边框：冲突

## 5. 机器管理模块

### 5.1 查看机器列表

1. 点击左侧导航栏「⚙️ 机器」
2. 卡片式布局展示所有机器
3. 状态指示灯：
   - 🟢 绿色：运行中
   - ⚪ 灰色：空闲
   - 🟡 黄色：维护中
   - 🔴 红色：故障

4. 每个卡片显示：
   - 机器编号和名称
   - 机器类型
   - 产能系数
   - 当前负载率
   - 今日排程数量

### 5.2 新增机器

1. 点击「+ 新增机器」按钮
2. 填写信息：
   | 字段 | 必填 | 说明 |
   |------|------|------|
   | 机器编号 | ✅ | 如：MCH-007 |
   | 机器名称 | ✅ | 如：加工中心VMC1270 |
   | 机器类型 | ✅ | 如：数控车床、加工中心、磨床等 |
   | 状态 | ✅ | 运行中/空闲/维护中/故障 |
   | 产能系数 | ✅ | 默认1.0，越高表示效率越高 |
   | 描述 | ❌ | 机器参数、规格等说明 |
3. 点击「保存」

### 5.3 管理机器状态

1. 找到目标机器卡片
2. 点击「状态」下拉按钮
3. 选择新的状态
4. 状态变更会影响排程：
   - 维护中/故障的机器无法安排新排程
   - 正在维护的机器上的排程需要重新安排

## 6. 产品工时配置模块

### 6.1 产品列表

1. 点击左侧导航栏「🏷️ 产品」
2. 表格展示所有产品
3. 显示字段：产品编码、产品名称、单位工时、单位、描述

### 6.2 配置工时

1. 点击「+ 新增产品」或「编辑」
2. 设置**单位工时**：生产单件产品所需的小时数
3. 这个工时是计算订单预估工时的基础

> 📐 工时计算公式：
> 订单预估工时 = 生产数量 × 产品单位工时 ÷ 机器产能系数

**示例：**
- 产品A单位工时：0.5小时/件
- 订单数量：100件
- 机器产能：1.0
- 预估工时 = 100 × 0.5 ÷ 1.0 = 50小时

## 9. 最佳实践

### 9.1 排程建议

1. **优先处理紧急订单**：先安排红色（紧急）和橙色（高优先级）订单
2. **合理安排机器负载**：避免单台机器负载过高，建议负载率控制在 85% 以内
3. **预留缓冲时间**：在订单之间预留 15-30 分钟缓冲，应对突发情况
4. **考虑设备维护**：提前查看机器维护计划，避开维护时间
5. **对齐工作时间**：默认按24小时排程，可根据实际班次调整

### 9.2 多用户协同

1. **分工明确**：建议按产品类型或机器类型划分计划员职责
2. **及时沟通**：发现冲突时及时与相关计划员协调
3. **操作规范**：修改他人排程前先沟通确认
4. **定期检查**：每天上班后检查当天排程是否有变动

### 9.3 数据维护

1. **每日备份**：系统自动备份，定期验证备份可用性
2. **定期清理**：归档已完成超过 3 个月的排程数据
3. **工时校准**：每季度根据实际生产数据校准产品工时

## 10. 常见问题

### Q1: 为什么排程保存失败？
**A:** 检查是否有时间冲突，红色边框表示有冲突，需要调整时间。

### Q2: 如何批量导入订单？
**A:** 目前系统暂不支持批量导入，如需此功能请联系管理员开发。

### Q3: 排程创建后可以修改吗？
**A:** 可以，直接在甘特图上拖拽调整，或点击任务块在右侧面板修改。

### Q4: 忘记密码怎么办？
**A:** 联系管理员重置密码，管理员在用户管理中可以重置密码。

### Q5: 甘特图加载很慢怎么办？
**A:** 可能是数据量太大，可以缩小时间范围查看，或联系管理员优化数据库。

### Q6: 为什么机器无法选择？
**A:** 检查机器状态，维护中和故障状态的机器无法安排排程。

### Q7: 如何导出排程报表？
**A:** 在甘特图页面点击右上角「导出」按钮，可导出 Excel 或 PDF 格式。

### Q8: 系统提示"乐观锁冲突"是什么意思？
**A:** 表示该排程已被其他用户修改，请刷新页面获取最新数据后再操作。
